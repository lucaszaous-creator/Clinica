# UI Kit — App de Faturamento (desktop WPF)

Recriação interativa do shell do app: sidebar verde + telas Pendências, Pacientes, Novo atendimento, Relatórios, e modais (Aviso de pendências, Dar baixa, Registrar glosa). Abra `index.html`. Telas em `screens.jsx` e `modals.jsx`; compõem os componentes do bundle (`Sidebar`, `Card`, `DataTable`, `KpiCard`, `AlertBanner`, etc.). Telas do app não recriadas aqui (Consultar guias, Agenda, Faturados, Glosas, Guias TISS, Parâmetros) exibem um aviso de "não recriado" em vez de invenção.
